## Oralfacial pain 



```mermaid


graph TD
    A[開始: 口腔顏面疼痛主訴] --> B(疼痛的發作模式?);

    %% 陣發性疼痛路徑
    B -- 陣發性 (Episodic): 突發, 數秒到數小時 --> C(疼痛性質?);
    C -- 尖銳、電擊痛 (Sharp, Electric Shock) --> D(Trigger Zone?);
    D -- 是 (有聖人模式 Refractory Period) --> E(疼痛在 V2/V3 或耳咽部?);
    E -- V2/V3 支配區--> F([ TN]);
    E -- 扁桃體/耳咽部 --> G([GN ]);
    D -- 否  --> H("伴隨同側自主神經症狀(流淚, 鼻塞)?");
    H -- 是 ( Stabbing Pain) --> I([Cluster Headache ]);
    H -- 否 (搏動性 Throbbing) --> J([Migraine ]);

    %% 持續性疼痛路徑
    B -- 持續性 (Continuous) --> K(疼痛性質?);
    K -- Burning, Tingling--> L(口腔黏膜外觀正常?位置？);
    L -- 正常 (多部位, 舌頭前2/3) --> M([ BMS ]);
    L -- 正常 (上顎後牙區/無牙脊) --> N([持續性特發性面部疼痛, PIFP ]);
    K -- 鈍痛, 酸痛 (Dull, Aching) --> O(下顎功能活動有關?);

    %% 軀體深層痛路徑
    O -- 是 (咀嚼, 開口) --> P(敲診敏感?);
    P -- 否 (耳前區, 顳部) --> Q([TMD ]);
    P -- 否 (顳動脈處, 顎跛行) --> R(顳動脈炎, Giant Cell Arteritis );
    P -- 是 (通常可定位, 較輕微) --> S([Periodontal Pain ]);
    
    O -- 否 --> T(疼痛是否因溫度加重?);
    T -- 是 (難定位) --> U([ Pulpal Pain ]);
    T -- 否 (上顎後牙區, 低頭加劇) --> V([Chronic Sinusitis ]);

    


```
