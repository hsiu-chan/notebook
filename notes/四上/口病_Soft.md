# Soft tissue

- 最常見sacroma: Liposarcoma
- 常見的週邊神經腫瘤: Neurofibroma
- GFAP-: Palisaded encapsulated neuroma (PEN)


- Sunburst 骨膜反應
  - osteosarcoma
  - hemengioma
  - vascular malformation
  - 地中海型貧血、鐮刀型貧血


- Schwann
  - Schwannoma
  - Palisaded encapsulated neuroma (PEN)
  - Granular cell tumor(GCT)

  3. 軟組織:Undifferentiated pleomorphic sarcoma, fibrosarcoma

## Reactive Lesions (反應性病灶)

| 病名 | 年齡/性別 | 病因/位置 | 特徵 | 組織 |
| --- | --- | :--- | --- | :--- |
| **Irritation Fibroma**<br>(刺激性纖維瘤) | 中年 | \oneline{
  <br>1. 創傷<br>2. 沿著咬合的頰黏膜<br>} | 最常見的軟組織腫瘤 | \oneline{<br>1. 結節狀 Collagen 堆積 → hyalinized<br>2. Atrophy of the rete ridges (若 rete ridges 多則稱為 fibrous epithelial polyp)<br>} |
| **Epulis Fissuratum**<br>(前庭區腫裂齦瘤) | 中老年女 | \oneline{<br>1. 與義齒相關<br>2. 前方、頰側齒槽嵴<br>} | 堅硬、纖維狀 | \oneline{<br>1. 纖維結締組織增生<br>2. 上皮過度角化 → rete ridge 增生、偽上皮增生<br>3. 發炎細胞浸潤<br>} |
| **Inflammatory Papillary Hyperplasia**<br>(炎性乳頭增生) | |^| | \oneline{<br>1. 義齒下方硬顎<br>2. 24hr 配戴義齒<br>3. 口腔衛生差(念珠菌)<br>} | \oneline{<br>1. 乳頭狀增生 → 鵝卵石狀<br>2. 乳頭狀紅斑<br>} | \oneline{<br>1. 偽上皮增生<br>2. 發炎細胞浸潤<br>} |
| **Papillary Endothelial Hyperplasia**<br>(Masson's tumor) | 成年人 | \oneline{<br>1. 血管內的內皮增生<br>2. 淺層軟組織(指頭、下嘴唇)<br>3. 血栓再通/創傷<br>} | \oneline{<br>1. 小而堅實的紅色/藍色腫塊<br>2. 長很快 (像 Angiosarcoma)<br>} | \oneline{<br>1. 血管內病變 → 殘餘血管壁、含大量血栓<br>2. 內皮細胞向內生長 → 假乳頭狀瘤 (單層細胞)<br>3. 相互吻合的血管通道<br>4. 染色: CD31, CD34, FLI1, vWF<br>} |
| **Oral Focal Mucinosis** | 年輕女性 | 牙齦 | \oneline{<br>1. Fibroblasts 過度分泌玻尿酸<br>2. 無痛結節腫大<br>} | \oneline{<br>1. 無 capsule、界線清楚、疏鬆黏液狀組織<br>2. 扁平的 rete ridges<br>3. 紡錘形 fibroblast<br>} |

## Gingival Lesions (牙齦病灶比較)

| \diagonal{特徵|病名} | **Pyogenic Granuloma (PG)** | **Peripheral Giant Cell Granuloma (PGCG)** | **Peripheral Ossifying Fibroma (POF)** |
| :--- | :--- | :--- | :--- |
| **好發族群** | 小孩、年輕人、女 (孕婦) | 中老年、女 (Youth mostly) | 年輕人 (Teenager/Young adult)、女 |
| **位置** | 牙齦 (75%)、舌、唇、頰 | **只在牙齦** (下顎後牙區多) | **只在牙齦** (上顎前牙區多, Facial 側) |
| **病因** | 創傷、荷爾蒙、口腔衛生 | 創傷、拔牙後 | 源於 PDL 或骨膜 (PG 纖維化/鈣化而來) |
| **臨床特徵** | \oneline{<br>1. 無柄有梗<br>2. 紅色/紫色腫塊<br>3. 懷孕瘤 (Pregnancy tumor)<br>} | \oneline{<br>1. 藍紫色腫塊 (含血鐵素)<br>2. 骨頭吸收 (Cupped-out)<br>} | \oneline{<br>1. 堅硬 (含鈣化)<br>2. 顏色較粉紅<br>3. 從 interdental papilla 長出<br>} |
| **組織學** | \oneline{<br>1. Lobular capillary hemangioma<br>2. 血管高度增生<br>3. 發炎細胞浸潤<br>} | \oneline{<br>1. 多核巨細胞 (Osteoclast-like)<br>2. 血色素沉積 (Hemosiderin)<br>3. 血管豐富<br>} | \oneline{<br>1. 纖維組織增生<br>2. 礦化物質 (骨頭、水泥質樣)<br>} |

## 神經與肌肉相關良性腫瘤 (Benign Tumors)

| 病名 | 年齡/位置 | 特徵 | 組織/染色 |
| :--- | :--- | :--- | :--- |
| **Traumatic Neuroma**<br>(創傷性神經瘤) | \oneline{<br>1. 中年<br>2. Mental foramen, 舌, 下唇<br>} | \oneline{<br>1. 神經受傷後的反應性增生<br>2. 痛覺/感覺異常<br>} | \oneline{<br>1. Nerve bundle 散亂增生<br>2. 很少發炎細胞<br>} |
| **Palisaded Encapsulated Neuroma (PEN)** | \oneline{<br>1. 中老年<br>2. 臉部(鼻)、硬顎、牙齦<br>} | \oneline{<br>1. 堅實無痛結節<br>2. 非 True neoplasm<br>} | \oneline{<br>1. Encapsulated<br>2. Schwann cells 柵狀排列<br>3. 可見 Axon (Silver stain+)<br>4. S-100(+), GFAP(-)<br>} |
| **Schwannoma**<br>(神經鞘瘤) | \oneline{<br>1. 青壯年<br>2. 舌頭<br>3. NF2 基因 (Merlin)<br>} | \oneline{<br>1. 生長慢、有 Capsule<br>2. 將神經往旁邊推<br>} | \oneline{<br>1. Antoni A: 柵狀排列, Verocay bodies<br>2. Antoni B: 散亂<br>3. S-100(+), Sox10(+)<br>} |
| **Neurofibroma**<br>(神經纖維瘤) | \oneline{<br>1. 年輕人<br>2. 皮膚、舌頭、頰黏膜<br>3. NF1 基因<br>} | \oneline{<br>1. 最常見週邊神經腫瘤<br>2. 無 Capsule (界線清楚)<br>3. 從神經中心長出<br>} | \oneline{<br>1. 混合 Schwann cell, fibroblast, mast cell<br>2. 逗號狀/波浪狀細胞核<br>3. S-100 局部(+)<br>} |
| **Granular Cell Tumor**<br>(顆粒細胞瘤) | \oneline{<br>1. 中年女性<br>2. 舌背 (最常見)<br>} | \oneline{<br>1. 源於 Schwann cell<br>2. 黃色結節<br>} | \oneline{<br>1. Granular cells (Lysosome 堆積)<br>2. 偽上皮增生 (PEH)<br>3. S-100(+), CD68(+)<br>} |
| **Congenital Epulis**<br>(先天性牙齦瘤) | \oneline{<br>1. 新生兒 (女嬰)<br>2. 上顎前牙區<br>} | \oneline{<br>1. 有梗息肉<br>2. 出生即有<br>} | \oneline{<br>1. 像 GCT 但無 PEH (上皮萎縮)<br>2. S-100(-), HLA-DR(+)<br>} |
| **Leiomyoma**<br>(平滑肌瘤) | \oneline{<br>1. 男性 (口內)<br>2. 唇、頰、舌<br>} | 血管肌瘤 (Angiomyoma) 最常見 | \oneline{<br>1. Spindle cells 束狀排列<br>2. 雪茄狀細胞核<br>3. SMA(+), Desmin(+)<br>} |
| **Rhabdomyoma**<br>(橫紋肌瘤) | \oneline{<br>1. 中年男性 (Adult type)<br>2. 頭頸部<br>} | \oneline{<br>1. 罕見<br>2. 淺層軟組織<br>} | \oneline{<br>1. 多角形細胞 (Spider web cells)<br>2. 橫紋 (PTAH stain)<br>3. Desmin(+), Actin(+)<br>} |

## 神經瘤比較表 (Key Comparison)

| \diagonal{特徵|病名} | **Neurofibroma** | **Schwannoma** | **PEN** |
| :--- | :--- | :--- | :--- |
| **成因** | NF1 (Chr. 17) | NF2 (Chr. 22) | Trauma (Reactive) |
| **細胞來源** | Fibroblast + Schwann | 只有 Schwann | Schwann |
| **Capsule** | 無 (Poorly defined) | 有 (Well defined) | 有 (Encapsulated) |
| **與神經關係** | 從神經內長出 (有 axon) | 把神經推開 (無 axon) | 有 Axon (受傷神經) |
| **組織特徵** | Mast cell, Wavy nuclei | Antoni A & B, Verocay bodies | 柵狀排列, 無 Antoni B |
| **染色** | S-100 (局部+) | S-100 (強+) | S-100 (+) |

## 血管病變 (Vascular Lesions)

| 分類 | 病名 | 特徵 | 組織/備註 |
| :--- | :--- | :--- | :--- |
| **Tumor** | **Infantile Hemangioma**<br>(嬰兒血管瘤) | \oneline{<br>1. 出生後出現 (草莓狀)<br>2. 會自行消退<br>3. GLUT-1 (+)<br>} | \oneline{<br>1. 內皮細胞增生<br>2. Cellular stage: 團塊狀<br>} |
| **Malformation** | **Vascular Malformation**<br>(血管畸形) | \oneline{<br>1. 出生即有<br>2. 不會消退，隨身體長大<br>3. Port-wine stain (GNAQ 突變)<br>} | \oneline{<br>1. 血管擴張但無內皮增生<br>2. GLUT-1 (-)<br>3. 包含動靜脈畸形 (Bruit, Thrill)<br>} |
| **Lymphatic** | **Lymphatic Malformation**<br>(淋巴管瘤) | \oneline{<br>1. 兩歲前出現<br>2. 舌頭 (Macroglossia)<br>3. 像青蛙下蛋 (透明囊泡)<br>} | \oneline{<br>1. 淋巴管擴張<br>2. D2-40 (+)<br>} |
| **Adult** | **Angiosarcoma**<br>(血管肉瘤) | \oneline{<br>1. 老年人 (頭皮、臉)<br>2. 預後差<br>3. 像瘀青但會腫大<br>} | \oneline{<br>1. 血管吻合 (Anastomosis)<br>2. 內皮細胞異型性 (Pile up)<br>3. CD31(+), CD34(+)<br>} |
| **Adult** | **Kaposi Sarcoma**<br> | \oneline{<br>1. HHV-8 病毒相關<br>2. AIDS 患者常見<br>3. 好發：硬顎、牙齦<br>} | \oneline{<br>1. Spindle cells<br>2. Slit-like vascular spaces<br>3. 紅血球外滲 (Extravasated RBCs)<br>} |

## 纖維與軟組織腫瘤 (Fibrous & Adipose)

| 病名 | 年齡/位置 | 特徵 | 組織 |
| :--- | :--- | :--- | :--- |
| **Nodular Fasciitis**<br>(結節性筋膜炎) | 年輕人 | \oneline{<br>1. 生長極快 (似肉瘤)<br>2. 具自限性 (Self-limiting)<br>3. MYH9-USP6 基因<br>} | \oneline{<br>1. Feathery pattern<br>2. 豐富的 Myofibroblasts<br>3. Extravasated RBCs<br>} |
| **Fibromatosis**<br>(Desmoid tumor) | 兒童、年輕人 | \oneline{<br>1. 局部侵犯性強 (Aggressive)<br>2. 不會轉移<br>3. 復發率高<br>} | \oneline{<br>1. 束狀 Fibroblasts<br>2. Beta-catenin (CTNNB1)<br>} |
| **Lipoma** | 成人 | 最常見的間葉瘤 | \oneline{<br>1. 成熟脂肪細胞<br>2. 浮在水面 (浮力)<br>} |
| **Liposarcoma**<br>(脂肪肉瘤) | 老年人 | \oneline{<br>1. 最常見的軟組織肉瘤<br>2. 好發深層 (大腿、腹膜)<br>} | \oneline{<br>1. Lipoblasts (及各種分型)<br>2. MDM2, CDK4 (+)<br>} |

## 症候群 (Syndromes)

| 症候群 | 基因 | 臨床特徵 |
| :--- | :--- | :--- |
| **Neurofibromatosis Type 1 (NF1)** | NF1 (Chr. 17) | \oneline{<br>1. 多發性 Neurofibroma<br>2. 咖啡牛奶斑 (Café-au-lait)<br>3. 腋下雀斑 (Crowe's sign)<br>4. Lisch nodules (虹膜)<br>} |
| **MEN 2B** | RET (Chr. 10) | \oneline{<br>1. 甲狀腺髓質癌 (Medullary thyroid ca.)<br>2. 嗜鉻細胞瘤 (Pheochromocytoma)<br>3. 口腔黏膜 Neuromas (Marfanoid body)<br>} |

## Drug-related gingival hyperplasia
- 停藥會消退的：Cyclosporine、鈣離子阻斷劑(nifedipine) 
- 停藥不會消退的：Phenytoin(抗癲癇藥物)，原則上只能靠 Gingivectomy
